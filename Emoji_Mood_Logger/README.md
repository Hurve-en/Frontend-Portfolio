# 🌸 Aesthetic Emoji Mood Logger UI  
A modern, soft, minimalist web-based UI for logging daily moods using emojis.  
Designed for calm, aesthetic UX with smooth animations, pastel color themes, and emotional expressiveness.

---

## ✨ Features

### 🎭 Mood Logging
- Tap an emoji to record your mood for the day  
- Soft, rounded emoji cards with hover animations  
- Option to add a short journal note  
- Edit today’s mood anytime

### 🗓️ Mood History
- Beautiful calendar-based view  
- Emoji displayed on each day  
- Filter moods by emoji  
- Optional mood heatmap  
- Mood trend line graph (7-day / 30-day)  
- View journal notes for past days

### 📝 Add/Edit Mood Modal
- Smooth pop-up modal  
- Emoji selector  
- Notes field  
- Blurred overlay background  
- Save / Update buttons

### ⚙️ Settings Page
- **Themes:** light, dark, pastel, neon  
- Manage mood emojis (add/remove categories)  
- Toggle charts on/off  
- Adjustable font sizes  
- Smooth transitions between pages

### 📱 Responsive Design
- Fully responsive for mobile, tablet, and desktop  
- Clean spacing and consistent typography  
- Floating “Add Mood” button for quick entry  
- Smooth page transitions & micro-interactions

---

## 🎨 UI/UX Style Guide

### Colors
- Pastel gradients  
- Muted tones  
- Low-contrast shadows  
- Soft borders and rounded corners (12–20px)

### Typography
- **Poppins**, **Inter**, or **DM Sans**  
- Calm, friendly weight selections  
- Clear hierarchy with comfortable spacing

### Microinteractions
- Scale animations (1.05 on hover)  
- Light shadows  
- Fade transitions for modals  
- Smooth page slide/opacity changes
